## 🎯 Quick Start Guide

### **First Time Using the System?**

```
STEP 1: Login to the System
        ↓
STEP 2: Navigate to Settings Page
        → Location: Main Menu → Settings
        ↓
STEP 3: Configure Your Organization
        → Add Ranks (Settings → Profile Settings → RK)
        → Add Companies (Settings → Profile Settings → Coy)
        → Create Leave Types (Settings → Duty Settings → Lve type)
        → Create Users (Settings → User Management → Create User)
        ↓
STEP 4: Add Soldiers
        → Go to Soldier Management
        → Create soldier profiles with personal info
        → Assign ranks and companies
        ↓
STEP 5: Create Duties
        → Go to Duty Management
        → Create duty with required soldiers
        → Assign soldiers (fixed or roster-based)
        ↓
STEP 6: Process Leave Requests
        → Soldiers submit leave requests
        → You approve/reject in Leave Management
        ↓
STEP 7: Generate Reports
        → Go to Reports
        → Run standard reports or create custom ones
        → Export as needed
```
